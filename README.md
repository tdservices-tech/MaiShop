# 🛍️ MaiShop – Boutique en ligne de sacs et chaussures pour femmes

![MaiShop](https://tdservices-tech.github.io/MaiShop/images/sac-beige-maishop.jpg)

## 📌 Description

**MaiShop** est une boutique en ligne spécialisée dans la vente de **sacs à main, sacs à dos, pochettes et tote bags** pour femmes, basée à **Abidjan, Côte d'Ivoire**.

Le site permet aux clientes de :
- Découvrir les collections de sacs
- Commander facilement via WhatsApp
- Recevoir leurs commandes à Abidjan et dans toute la Côte d'Ivoire
